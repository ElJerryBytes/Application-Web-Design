## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero creado por John Gruber en 2004. Su principal objetivo es permitir que las personas escriban texto con formato de manera sencilla, utilizando una sintaxis simple que se puede leer fácilmente incluso sin convertirla.

### Principales usos de Markdown:

1. **Documentación de proyectos**: Es el estándar para archivos README en plataformas como GitHub, GitLab y Bitbucket.

2. **Creación de contenido web**: Muchos blogs, wikis y sistemas de gestión de contenido utilizan Markdown por su simplicidad.

3. **Toma de notas**: Aplicaciones como Notion, Obsidian, Typora y Bear utilizan Markdown como formato principal.

4. **Escritura técnica**: Documentación de software, tutoriales y manuales técnicos se escriben frecuentemente en Markdown.

5. **Formato de texto**: Permite dar formato sin necesidad de botones o menús complejos. Por ejemplo:
   - *Cursiva* se logra con asteriscos: `*texto*`
   - **Negrita** se logra con doble asterisco: `**texto**`
   - [Enlaces](https://ejemplo.com) se crean con corchetes y paréntesis

Markdown es especialmente popular entre desarrolladores y escritores técnicos porque permite enfocarse en el contenido sin distraerse con herramientas de formato visual complejas. Además, al ser texto plano, es portable y puede controlarse fácilmente con sistemas de control de versiones como Git.
